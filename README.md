## ✨ Features

- 🎵 Full media controls (play, pause, next, previous)
- 🔁 Like, repeat & seekbar support
- 🔒 Lock screen media controls
- 🎛️ Control panel integration
- ⚡ Optimized performance (smooth experience)
- 📱 Clean & minimal UI
- 🔗 Open music links directly in the app
